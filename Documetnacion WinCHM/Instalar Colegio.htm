Para poder instalar el sistema colegio debe cumplir con la siguiente condiciones

  1. Conexion de red con protocolo TCP/IP
  2. Verificar conectividad con servidor del sistema: SERVIDOR-BD
  3. Verificar que se encuentre intalado el Microsoft® .NET Framework 2.0  y  Crystal Reports® para Microsoft .NET Framework 2.0

una vez revisada e instalado los programas anteriores con el instaldor de iColegio instalar el programa en el computador .
