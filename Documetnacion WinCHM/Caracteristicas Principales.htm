El sistema colegio es una aplicación para administración académica que cuenta con módulos para las administración de :

  • Matricula
    • Expedientes de Estudiantes
    • Generación de Grupos y  Listas de Clase
    • Registro de grupos centralizado por : años, niveles, grupos
  • Cobros
    • Generación de cobros
    • Pago por Adelantado
    • Reportes
      • Estados de Cuenta
      • Ingresos
    • Gestión con Bancos
  • Control de Notas
    • Control de Ausencias
    •  Generación de archivo de notas para profesores
    • Carga de notas por profesor via archivo Excel
    • Administracion de Notas Centralizado por : años, niveles, grupos
